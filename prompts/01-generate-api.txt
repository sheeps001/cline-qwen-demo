Создай Express сервер с CRUD операциями для пользователей.
Используй массив в памяти как БД.
Добавь валидацию email и пароля (минимум 6 символов).

Структура:
- POST /users - создание
- GET /users - список всех
- GET /users/:id - получить одного
- PUT /users/:id - обновить
- DELETE /users/:id - удалить

Добавь обработку ошибок и правильные HTTP статусы.
