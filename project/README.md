# Express Users API

Это REST API приложение для управления пользователями, разработанное с использованием Express.js.

## Описание

Проект представляет собой простое API для выполнения CRUD операций над пользователями. Он использует массив в памяти в качестве базы данных и включает валидацию данных с помощью Joi. Приложение также включает middleware для логирования запросов и модульную систему обработки ошибок.

## Установка

1. Клонируйте репозиторий:
```bash
git clone <url>
cd <project-directory>
```

2. Установите зависимости:
```bash
npm install
```

3. Создайте файл `.env` на основе `.env.example`:
```bash
cp .env.example .env
```

4. Запустите приложение:
```bash
npm start
```

Приложение будет доступно по адресу `http://localhost:3000`.

## API Эндпоинты

### Получить всех пользователей
```
GET /users
```

### Получить пользователя по ID
```
GET /users/:id
```

### Создать нового пользователя
```
POST /users
```

Тело запроса должно содержать:
```json
{
  "name": "Имя пользователя",
  "email": "email@example.com",
  "age": 30
}
```

### Обновить пользователя
```
PUT /users/:id
```

Тело запроса должно содержать:
```json
{
  "name": "Новое имя",
  "email": "newemail@example.com",
  "age": 25
}
```

### Удалить пользователя
```
DELETE /users/:id
```

## Валидация

При создании и обновлении пользователей применяется валидация:
- Имя: строка длиной от 3 до 30 символов, обязательно
- Email: действительный email адрес, обязателен
- Возраст: целое число от 0 до 120, обязательно
