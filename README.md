# 🚀 Cline + Qwen OAuth Demo

Данный проект демонстрирует настройку и использование AI-агента Cline в Visual Studio Code с использованием Qwen OAuth для разработки Node.js приложения.

---

## 📌 О проекте

В рамках проекта была выполнена настройка среды разработки с использованием искусственного интеллекта.  
Основная цель — показать, как AI может ускорять процесс написания кода, его анализа и тестирования.

---

## ⚙️ Используемые технологии

- Node.js
- Express
- Visual Studio Code
- Cline (AI-агент)
- Qwen Code (AI модель)
- GitHub

##🤖 Возможности Cline

В ходе работы были продемонстрированы возможности AI-агента:
	•	Генерация кода
	•	Рефакторинг
	•	Объяснение кода
	•	Создание тестов
	•	Генерация документации

⸻

##🔄 Режимы работы Cline

📝 Plan
	•	Объясняет и предлагает решения
	•	Не изменяет файлы

⚡ Act
	•	Создаёт и редактирует файлы
	•	Выполняет действия автоматически

⸻
